# LoadingButton
带loading效果的button

## How To
### Step 1. Add the JitPack repository to your build file
```groovy
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
### Step 2. Add the dependency
```groovy
dependencies {
	        compile 'com.github.abcdqianlei1990:LoadingButton:1.2'
	}
```
